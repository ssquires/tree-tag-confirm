Directions

The purpose of this HIT is to confirm the location of a tree that another mTurker has marked on a map, and mark whether the base of the tree is blocked by anything in the image below.

1. The red square in the image below should start out close to the base of a tree. Click and drag the image so that the red square is perfectly centered on the base of the tree. Make sure to use the tree whose base starts out closest to the red square.

2. If you do not see a tree near the red square, or if the base of the tree is blocked by another object (like a car), indicate that there is a problem by checking the appropriate checkbox below. You may also click "Not a Tree" if no image of the tree appears.

3. Check one of the two options: "Public" if the tree is on public land (between the sidewalk and the street), or "Private" if the tree is on someone's property. For examples of each, see the bottom of the page.

Use the "Submit HIT" button at the bottom of the page when you're done.
